# 🎮 24点游戏说明书

## 🥅 游戏目标

通过加减乘除四则运算, 将给出的数字计算为 **24**.

## 📖 例题

1. 给定 **(4,4,7,7)** 四个数, 则可以通过四则运算式 **7÷(4-4÷7)** 算出 **24**.
2. 给定 **(1,7,13,13)** 四个数, 则可以通过四则运算式 **(13×13-1)÷7** 算出 **24**.
3. 给定 **(1,1,1,8,12)** 五个数, 则可以通过四则运算式 **1÷(1÷8-1÷12)** 算出 **24**.
4. 给定 **(4,9,9,11,11)** 五个数, 则可以通过四则运算式 **(((11+9)×11)-4)÷9** 算出 **24**.
5. 给定 **(9/5,4/3,4/3,1/9)** 四个数, 则可以通过四则运算式 **9/5×(4/3+(4/3÷1/9))** 算出 **24**.
6. 给定 **(4+4i,1,3+2i,3+3i))**, 则可以通过四则运算式 **(4+4i)÷(1-(3+2i)÷(3+3i))** 算出 **24**.

是不是很简单?

## 👆 操作方法

- **计算:** 点击第一个数字 → 点击运算符 → 点击第二个数字. 
- **撤销:** 点错了可以撤销.
- **恢复:** 撤销过头可以恢复.
- **重做:** 重新开始当前题目.


## 💡 功能按钮
- **尝试:** 这是只能提示, 可以将两个可以先进行运算的数标成粉色.
- **结构:** 只显示算式括号结构, 不显示具体数字. 可以长按复制文本.
- **答案:** 直接显示参考答案. 可以长按复制文本.
- **分享:** 复制当前题目文本, 考考你的朋友!
- **跳过:** 就是跳过.

## 📂 侧边菜单
点击 **📖 游戏说明书** 即可打开说明书 (就是你正在看的).

点击 **☁ 从 Github 更新题库**, 则可以从 GitHub 下载最新题库 
- 可能需要翻墙.

点击 **⚙️ 模式设定**, 则可以修改游戏模式.
- 避免纯加减: 解答中必须含有乘法或除法.
- 必须有除法: 字面意思.
- 避免平凡乘法: 最后一步不能是 3×8 这类.
- 包含分数加减: 算式包含分式加减.
- 除法风暴: 解答使用 n-2 次除法, 此处 n 是数字个数.

点击 **🧮 24点计算器**, 则可以打开计算器.
- 输入小于等于 5 个数, 即可求解.
- 复数输入格式: 2+J, J, 3-I2, 3-2i. 请不要在加号或减号左右加空格.
- 分数输入格式: 2/3. 请不要在分数线左右加空格.
- 解的算式是经过正规化的. 例如, a + b 与 b + a 视作同一算式, a / (b / c) 与 (a * c) / b 也视作同一算式.

点击 **🎲 随机模式**, 则可以进入休闲出题模式
